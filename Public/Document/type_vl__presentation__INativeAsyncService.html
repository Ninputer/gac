INativeAsyncService Class

Asynchronized operation service. GacUI is not a thread safe library except for this service. To access this service, use INativeController::AsyncService.

Member Functions:

NameDescription
InvokeAsyncInvoke a specified function with an specified argument asynchronisly.
InvokeInMainThreadInvoke a specified function with an specified argument in the main thread.
InvokeInMainThreadAndWaitInvoke a specified function with an specified argument in the main thread and wait for the function to complete or timeout.
IsInMainThreadTest is the current thread the main thread.
